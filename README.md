# Expense Tracker App Prototype
# For Programming On The Internet Assessment 1 Autumn 2026 UTS
This prototype webpage allows users to put expenses in to a tracker, which shows a list of the expenses. These expenses have several fields, and each one can be modified once entered in to the tracker. Expenses may also be deleted as well. There are some stats displayed at the bottom of the page for the user to be able to understand their expenses more.

# Technical stack:
Frontend: HTML+CSS / React
API: FastAPI
Backend: MySQL in python
Database: SQL relational database

# Feature list:
Add expenses
Delete expenses
Modify fields in expenses
See stats on expenses
Pie chart representing expenses
